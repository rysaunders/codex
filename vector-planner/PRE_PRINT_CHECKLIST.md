# Pre-Print Checklist

Use this before the first serious VECTOR test print.

## Digital Checks

- HTML builds successfully.
- PDF exports successfully if Playwright dependencies are installed.
- Page count matches the expected planner configuration.
- A5 page size is confirmed.
- Margins are inspected.
- Footers are visible.
- No text is clipped.
- Dot grid is subtle.
- Scorecards are readable.
- Stat panels are writable.

## Physical Checks

Print 4 sample pages first:

- Cover.
- Daily Orientation.
- Daily Execution.
- Weekly Review.

Inspect:

- Pen usability.
- Contrast.
- Binding-side margin.
- Duplex alignment.
- Whether pages feel too boxy or cramped.
- Whether prompts are useful but not overbearing.

## Print Settings

- Use A5 paper if available.
- Otherwise print scaled to fit on letter for prototype only.
- Use duplex long-edge binding.
- Use grayscale.
- Use actual size when printing on A5.
- Avoid browser shrink-to-fit if sending a print-ready PDF.
